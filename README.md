# liri-node-app

In this activity we will search Concerts and Movies

## Instructions

* When the completed 'liri-node-app' file is run with node, it should be able to handle command line arguments similar to those below:

  ```
  // Ex: Search for concert-this 98 degrees

  node liri-node-app show (Venue name, Location and Date)
Venue: Neal S. Blaisdell Center
Location: Honolulu
Date: 02/14/2020
  ```

  and

  ```
  // Ex: Search for movie-this boomerang

  node liri-node-app show (Title, Release Year, IMDB rating, rotten tomatoe rating, country, language, plot and actors)
  Title: Boomerang
Release Year: 1992
IMDB Rating: 5.5
Rotten Tomatoes: 44%
Country: USA
Language: English, French
Plot: A successful executive and womanizer finds his lifestyle choices have turned back on him when his new female boss turns out to be an even bigger deviant than he is.
Actors: Eddie Murphy, Robin Givens, Halle Berry, David Alan Grier
  ```

* In these examples the liri-node-app expects input in the following format:

  ```
  node liri-node-app <concert-this> <movie-this>
  ```
